Herrería Web
Aplicación web desarrollada para una herrería, con el objetivo de mostrar nuestros trabajos, contar nuestra historia y facilitar el contacto con clientes mediante formularios y presupuestos en línea. El proyecto está construido con React, siguiendo buenas prácticas de desarrollo, componetización y estilos modernos.

Funcionalidades
- Galería de trabajos: muestra visual de proyectos realizados.
- Preguntas frecuentes (FAQ): implementadas con Bootstrap para un diseño claro y responsivo.
- Presupuestos en línea:
  - Formulario de contacto.
  - Formulario conectado a una API para calcular presupuestos en tiempo real.
- Botón de WhatsApp: fijo y visible en todas las páginas para contacto inmediato.
- Sección de contacto: enlaces directos a nuestras redes sociales.
- Animaciones y estilos personalizados: uso de CSS, media queries y diseño responsivo.
- Navegación con React Router: estructura de rutas y layouts organizados.
- Componetización: separación clara de código y estilos para mayor mantenibilidad.

Tecnologías utilizadas

- React.js  
- JavaScript (ES6+)  
- HTML5  
- CSS3  
- Bootstrap  
- React Router  
- API REST para presupuestos
